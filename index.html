Bienvenido/a.

Este portal le permite explorar los flujos de su nueva plataforma.

Note: Arrastre los diagramas con el ratón o utilice la rueda para hacer zoom (o pellizque en su pantalla táctil) para examinar cada detalle de manera fluida.

1 Mapa General del Sitio Web

Este diagrama muestra la estructura completa de la plataforma, dividida en las secciones principales: la zona pública para visitantes, el proceso de registro y acceso, las herramientas para los compradores y el panel de control del administrador. Sirve como un mapa general para entender cómo se conecta todo.

2 Registro de un Nuevo Usuario

Este diagrama detalla el paso a paso que debe seguir una persona para crear su cuenta. Incluye el llenado del formulario y el importante proceso de verificación de identidad mediante códigos de seguridad enviados a su email y teléfono, garantizando que solo usuarios reales puedan registrarse.

3 Proceso de Pago de Seña

Aquí se describe qué ocurre cuando un usuario nuevo intenta ofertar por primera vez. El sistema le solicitará una seña para habilitarlo. Se muestran las opciones para enviar el comprobante de pago (vía web o WhatsApp) y cómo el administrador revisa y aprueba dicha seña para que el usuario pueda comenzar a participar.

4 Sistema de Ofertas en Tiempo Real

Este es el corazón de la subasta. El diagrama explica cómo el sistema verifica que un usuario pueda ofertar, procesa su oferta asegurándose de que sea válida, y actualiza la pantalla de todos los participantes al instante. También incluye la "regla del último minuto", que añade tiempo extra si entra una oferta al final para hacer la puja más justa y emocionante.

5 Sistema de Avisos al Cliente

La comunicación es clave. Este diagrama muestra qué avisos automáticos recibirá el cliente y por qué medio (WhatsApp, Email o en la propia web). Esto incluye desde el código para registrarse, notificaciones de que su oferta fue superada, hasta el aviso de que ha ganado un lote.
